Locaudio
========
Sound source localization in reconfigurable wireless acoustic sensor networks

## Installing Dependencies
##### Dependencies
- Python 2.7
- Matplotlib
- Scipy
- Numpy
- SciKit-Learn
- Flask

##### To Install
`make depend` or `pip install -r requirements.txt`

## Running
##### To Generate Documentation
`make documentation`

##### To Run Tests
`make run_tests`

def locaudio.triangulation.determine_peaks (   opt_vals,
  label_list 
)

Given a list of "optimized" points and their corresponding probabilities and a list of labels returned from the clustering algorithm, this function goes through all of the optimized points and returns the ones with the highest probabilities and issues them as cluter centers.

This function is used to ensure that the center for each cluster also has the highest probability in the cluster of being the sound position.

Parameters
opt_valsA list of tuples where each element is a key-value pair where the key is the Point object of the optimization x and y position and the value is the associated probability
label_listA list of integers where the index of the list corresponds to a certain key-value pair in the first parameter list and the integer value represents which cluster it belongs to.

  l_ref,

Determines a list of possible positions of where the sound will be located.

These positions are determined by changing iterating through the list of node events and optimizing the probability density function with the initial guess being the position of the node event. Hopefully the optimization will find the local minima.

Parameters
rRefThe reference distance at which the reference sound pressure level was recorded
lRefThe reference sound pressure level used to determine the distance from the newly measured sound pressure level
initGuessThe initial guess for gradient decent
nodeEventsThe list ofassociated data when a node detects with some confidence that the sound has been identified
Returns
A list of the possible x and y positions of the sound.

def locaudio.triangulation.position_evaluation (   x,
  y,
  r_ref,
  l_ref,
  node_events 
)

Evaluation function to deterimine with some weight, where a sound is located in the mesh network of nodes.

Given an x and y, this returns a weight. The higher the weight, the higher the likelihood that the sound originated from x and y.

Parameters
xthe horizontal location of the node event when the node event was captured
ythe vertical location of the node event when the node event was captured
rRefThe reference distance at which the reference sound pressure level was recorded
lRefThe reference sound pressure level used to determine the distance from the newly measured sound pressure level
nodeEventsThe list of associated data when a node detects with some confidence that the sound has been identified
Returns
The a result, given independent variables, x and y, and a configuration of node events with per sample sound constants, rRef and lRef, will be a value representing the likelihood of the sample sound being located at position (x, y).

def locaudio.triangulation.set_node_events_std (   node_events)

Uses the relative time differences between the node events mixed with the confidence of sound recognition to determine the standard deviation of the node event.

Node events that are closest to the node event that has the largest time stamps will have a lower standard deviation. Also, node events that have a larger sound recognition confidence will be given lower standard deviations.

Parameters
nodeEventsThe list of associated data when a node detects with some confidence that the sound has been identified
